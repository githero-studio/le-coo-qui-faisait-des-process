# Le COO qui faisait des process

Tu viens d'être nommé COO d'une startup de 80 personnes qui a grandi trop vite. Jusqu'ici, tout fonctionnait "à l'oral". Les décisions se prenaient autour de la machine à café. Les process, c'était "on verra quand on sera plus grands". Vous êtes plus grands. C'est le chaos.

Tu arrives de chez McKinsey. Tu as des frameworks pour tout. Tu as un Notion template pour chaque situation. Tu as une matrice RACI tatouée sur l'avant-bras (métaphoriquement). Ta mission : mettre de l'ordre. Le problème : personne n'a demandé d'ordre.
